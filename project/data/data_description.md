# Описание используемых данных

В этом файле описаны **использованные данные**, которые используются для проекта.


## **Stanford Cars**

- **Описание датасета** 
    > Набор данных Stanford Cars содержит 16 185 изображений 196 классов автомобилей. Данные разделены на 8 144 обучающих изображения и 8 041 тестовое изображение, при этом каждый класс был разделён примерно поровну. Классы обычно соответствуют марке, модели и году выпуска, например Tesla Model S 2012 года или купе BMW M3 2012 года.
- **Лицензирование:** MIT
- **Использование в проекте:** Данные используются для обучения модели классификатора автомобилей

## **Udacity Self Driving Car**

- **Описание датасета** 
    > Набор данных содержит 97 942 метки для 11 классов объектов и 15 000 изображений. Имеется 1720 нулевых примеров (изображений без меток).
- **Лицензирование:** MIT
- **Использование в проекте:** Данные используются для обучения модели определять объекты на фото
