# Идеи тем для проекта по курсу «Инженерия Искусственного Интеллекта»

В этом файле находятся идеи тем для итогового мини-проекта по курсу. Идеи будут приводится со всей нужной информацией, как будто заполнение итоговой карты проекта. Файл будет пополняться в ходе нахождения новых интересных датасетов, и придумывания им интересной реализации.

---

## 1. Прогноз ежедневных продаж в магазине **(Стремление к [INTERMEDIATE])**

- **Краткое описание:**  
  **Идея:** 
    Сервис для прогнозирования количества продаж в магазине на следующие несколько дней
  **Вход:** 
    Данные о продажах
  **Выход:** 
    Прогноз продаж на N дней вперед
  **Сервис:** 
    API, возвращающее прогноз на заданный период
- **Ссылка на датасет:** [Store Item Demand Forecasting Challenge](https://www.kaggle.com/competitions/demand-forecasting-kernels-only/data)

---

## 2. Прогноз количества гостей в ресторане **(Стремление к [INTERMEDIATE])**

- **Краткое описание:**  
  **Идея:** 
    Помощь ресторану или кафе предсказать ежедневную посещаемость для оптимизации работы заведения
  **Вход:** 
    История ежедневных посещений (+календарь,погода)
  **Выход:** 
    Прогноз количества посетителей на N дней
  **Сервис:** 
    API, возвращающее прогноз на заданный период
- **Ссылка на датасет:** [Restaurant Visitor Forecasting](https://www.kaggle.com/competitions/recruit-restaurant-visitor-forecasting/data)

---

## 3. Детекция и классификация старых автомобилей по фото **(Стремление к [INTERMEDIATE])**

- **Краткое описание:**  
  **Идея:** 
    Сервис для любителей или оценщиков, который находит автомобиль на изображении и определяет его марку и модель (_акцент на классику!_)
  **Вход:** 
    Изображение с автомобилем
  **Выход:** 
    Bounding box с автомобилем и предсказание марки/модели
  **Сервис:** 
    API, возвращающее координаты bounding box и класс автомобиля
- **Ссылка на датасет:** [Stanford Cars Dataset](https://www.kaggle.com/datasets/jessicali9530/stanford-cars-dataset)
